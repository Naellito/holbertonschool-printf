
# Custom printf Implementation

## Overview

This project is part of the Holberton School curriculum and involves creating a custom implementation of the `printf` function in the C programming language. By replicating the core functionalities of the standard `printf`, we aim to gain deeper insights into:

- **String formatting**: Handling and manipulating strings for output.
- **Variadic functions**: Using functions that accept a variable number of arguments.
- **Low-level programming**: Writing efficient code close to the system.
- **Collaboration**: Developing high-quality, functional software as a team.

## Features

### Supported Format Specifiers:
- **`%c`**: Prints a single character.
- **`%s`**: Prints a string. If the string is `NULL`, it prints `"(null)"`.
- **`%d` / `%i`**: Prints an integer, including negative values, using a recursive helper function.
- **`%%`**: Prints the `%` character itself.
